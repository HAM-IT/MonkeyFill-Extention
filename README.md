## What Is MonkeyFill?

MonkeyFill is a Chrome extension that reads any job offer and instantly tells you:

- How well your CV matches it (with a real ATS score)
- What skills are missing from your application
- Exactly how to close the gap
- Then generates a tailored, downloadable CV in seconds

No accounts. No subscriptions. No nonsense. Just results.

## Installation

> MonkeyFill is in private beta — not on the Chrome Web Store yet.
> You install it from a folder on your computer. Takes 2 minutes.

**What you need:** Google Chrome (or Brave / Edge) + the ZIP file from the [Releases page](../../releases/latest)

### Step 1 — Download & Unzip
Download `MonkeyFill-Beta-v0.1.zip` from the Releases page and unzip it.
- **Mac:** Double-click the ZIP
- **Windows:** Right-click → Extract All

⚠️ Put the folder somewhere permanent (like Documents). Chrome needs it to stay there — if you move or delete it, the extension stops working.

### Step 2 — Open Chrome Extensions
Type this in your Chrome address bar and press Enter:
```
chrome://extensions
```

### Step 3 — Enable Developer Mode
Toggle **Developer Mode** ON in the top-right corner.
*(It just lets Chrome load extensions from your computer. Nothing scary.)*

### Step 4 — Load the Extension
Click **"Load unpacked"** → select the `MonkeyFill-Beta-v0.1` folder → click Open.

### Step 5 — Pin It
Click the 🧩 puzzle piece icon in Chrome → find MonkeyFill → click the 📌 pin.

✅ The 🐒 monkey icon is now in your toolbar. Click it to start.
